REM vs EM
difference and usage

REM and EM are both scalable units. Their values are always relative to the value of something else.

1rem = 16px

REM unit is only relative to the document's root element, while the EM unit is only relative to the immediate parent of the targeted element.

Therefore, in latter button example, while using EM, the padding will grow in relation to the btn

This can be used to scale everything in proportion to the text value
